Testing

These files are for testing the implementation of Sigma16. They also provide examples of all the assembly language statements and the instruction set.

  • AsmCodeGen Examples of each assembly language statement, including instructions and directives.

  • AsmErrMessages Examples of errors that are detectable by the assembler.

  • StoreLoop While loop containing a store instruction

  • Arithmetic Test arithmetic operations.

  • Logic Test the operations on bits, fields, and words.

  • Memory Loads, stores, stack operations, saving and restoring registers.

  • CmpJump Test the comparisons, jumps, and conditional jumps.

  • Performance A long nested loop that executes a large number of instructions, allowing measurements of the perfornace of an emulator or other implementation.

  • Modules: Mod1 Mod1 imports Mod2 and Mod3, Mod2 Mod1 imports Mod2 and Mod3, and Mod3 Mod1 imports Mod2 and Mod3

  • PrimarySecondary

  • RFwrite
