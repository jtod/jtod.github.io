Simple examples

  • ConstArith This is a good example to start with: a small program that does some arithmetic, illustrating the use of lea, add, and mul.

  • Add A small program that adds two variables, illustrating the use of load, store, and data.

  • LoadStore Demonstrate how to use the load and store instructions to access variables.

  • CmpJump Illustrate how to use the compare and conditional jump instructions to perform an if-then conditional.
