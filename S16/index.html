Sigma16 home page

Sigma16 is a computer architecture for at teaching and research.

Running the software

You can run Sigma16 in several ways:

  • Run the latest release in your browser. Not yet available; for now this link just runs the current pre-release. The first actual release of Version 3 is expected February 2020. Previous major versions of Sigma16 are available only on a private Moodle server.

  • Download the source files and visit index.html in a browser. After the download, you won’t need to be connected to the Internet, but a few features won’t work. When you open one of the example programs, you need to copy it and paste it into the Editor tab (the button “Copy example to editor” won’t work).

  • Build an executable compiled for your platform. The app will run faster, it has better ability to save files, and it doesn’t need access to the Internet.
