RRR format

An RRR instruction is written with an instruction name (menonic) and three register operands> For example, the "trap" instruction has mnemonic 13 (and the hex value of 13 is d), so the assembly language "trap R1,R2,R3" is translated to machine language as d123:
op d a b
d 1 2 3
0-3 4-7 8-11 12-15

EXP format

First word of the instruction
op d a b
0-3 4-7 8-11 12-15
Second word of the instruction
p q r s
0-3 4-7 8-11 12-15
